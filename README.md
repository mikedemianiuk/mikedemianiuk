## Hi, I'm Mike 👋

Most PMs read about payment rails. I built a simulator for them.

I'm a Payments PM building tools that make fintech genuinely learnable — not just memorizable. Everything here is something I use in my actual prep and day-to-day work. If you're studying for a fintech PM role or just want to understand how money actually moves, you're in the right place.

## What I Work On

- 🏦 **Payment rail simulation** — ACH, Wire, RTP, FedNow, cards, USDC, BTC, ETH, SOL in one platform
- 🤖 **AI PM interview coaching** — Claude-powered case interviewer with cross-session memory and 4-pillar scoring
- 📰 **Automated fintech intelligence** — 50+ RSS feeds → personalized weekly digest via Cloudflare Workers + Gemini
- 📊 **Case study library** — 25+ company-specific PM cases (Spade, Lithic, Unit, Adaptive, Coast, Capital One)
- 🔁 **Dual-ledger reconciliation** — simulating the gap between internal state and what the network actually reports
- ⚡ **Agentic commerce** — machine-initiated payments, metered billing, ACP protocol
- 🔐 **Fintech regulation** — OCC charter strategy, BaaS model risk, ISO 20022 migration
